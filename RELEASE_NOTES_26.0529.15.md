## 首次使用授权

本版本新增离线机器码授权。用户首次打开平台时，需要先完成激活：

1. 双击 `双击启动_平台.bat`。
2. 浏览器进入授权页面，复制页面显示的本机机器码。
3. 将机器码发送至 `1778140731@qq.com`，邮件中建议写明使用人或单位名称、授权期限需求。
4. 授权人生成授权码后，用户粘贴到页面并点击“激活并进入平台”。
5. 激活成功后，授权信息保存在本机 `.license\activation.lic`，后续同一台电脑无需重复输入。

说明：

- 授权码与机器码绑定。
- 机器码由当前电脑的系统信息、硬件标识、主机名等信息在本机摘要生成；用户无需手动查找硬件信息，只需复制授权页面显示的机器码。
- 更换电脑、重装系统或关键硬件信息变化后，需要重新生成授权码。
- 授权码生成器是发行人本地工具，不包含在用户发行包中，也不应上传到公开仓库。

## 版本内容

- 一键完成数据导入、变量配置、自动建模与 Train / Test 评估。
- 支持 SHAP、GeoShapley、SHAP-PDP 和 2D SHAP-PDP 可解释分析。
- 支持 SHP 空间映射、主导因子识别、SHAP 聚类和聚类空间分布。
- 支持历史记录、技术文档和完整结果包导出。
- 支持离线机器码授权，便于对外分发和项目归档。

## 包含内容

- Streamlit 建模平台主程序。
- 便携 Python 3.11 运行时。
- 已配置虚拟环境和核心依赖。

## 快速开始

1. 下载 `ML-Modeling-Platform_26.0529.15.zip`。
2. 解压整个 `ML建模平台` 文件夹，保持 `.python`、`.venv`、`assets`、`src` 等目录完整。
3. 进入文件夹后双击 `双击启动_平台.bat`。
4. 首次打开时按页面提示完成机器码授权。
5. 浏览器进入平台后，按“项目信息、数据导入、变量配置、一键建模、分析生成”的顺序使用。
